=head1 NAME

xdg-play - play an XDG event identifier

=head1 SYNOPSIS

B< xdg-play> [I<OPTIONS>] [--] I<EVENTID> [...]
 B<xdg-play> {B<-l>|B<--list>}    [I<OPTIONS>]
 B<xdg-play> {B<-w>|B<--which>}   [I<OPTIONS>] [--] I<EVENTID> [...]
 B<xdg-play> {B<-W>|B<--whereis>} [I<OPTIONS>] [--] I<EVENTID> [...]
 B<xdg-play> {B<-h>|B<--help>}    [I<OPTIONS>] [--] I<EVENTID> [...]
 B<xdg-play> {B<-V>|B<--version>}
 B<xdg-play> {B<-C>|B<--copying>}

=head1 OPTIONS

=head2 COMMAND OPTIONS

=over

=item B<-w>, B<--which>

For each I<EVENTID> argument, prints to standard output the full path of
the sound file that would have been played when this argument was
supplied to B<xdg-play(1)>.  It does this by searching for an event
identifier in the appropriate sub-directories of the C<XDG_DATA_DIRS>
and C<XDG_DATA_HOME> environment variables using the same algorithms as
L<xdg-play(1)>.

=item B<-W>, B<--whereis>

For each I<EVENTID> argument, locates the sound files for the specified
event identifiers.  The supplied names are first stripped of leading
path-name components and any (single) trailing extension of the form
F<.ext> (for example: F<.wav>).  B<xdg-play> then attempts to locate the
desired sound file in the standard Linux places, and in the places
specified by C<XDG_DATA_DIRS> and C<XDG_DATA_HOME>.

=item B<-l>, B<--list>

Output list of effective lookup paths that B<xdg-play> is using.

=item B<-h>, B<--help>

Prints usage information to standard output and exits.

=item B<-V>, B<--version>

Prints detailed version information and exits.

=item B<-C>, B<--copying>

Prints detailed copying permissions and exits.

=back

=head2 GENERAL OPTIONS

=over

=item B<-a>, B<--all>

Print all matching sound files, not just the first.

=item B<-o>, B<--skip-dot>

Skip directories in C<XDG_DATA_DIRS> that start with a dot.

=item B<-t>, B<--skip-tilde>

Skip directories in C<XDG_DATA_DIRS> that start with a tilde and entries
that reside in C<XDG_DATA_HOME>.

=item B<-O>, B<--show-dot>

If a directory starts with a dot and a matching event sound file was
found for that path, then print C<./eventid.ext> rather than the full
path.

=item B<-T>, B<--show-tilde>

Output a tilde when a directory matches the C<HOME> directory.

=item B<-s>, B<--theme> I<THEME>

Use the sound theme named I<THEME> when looking for sound files
corresponding to event identifiers.

=back
